# AI Crypto Brain

AI-powered crypto strategy monitoring and decision support system.

## Overview

AI Crypto Brain is a portfolio-level analysis engine designed to review multiple crypto trading strategies, compare their performance, and provide strategy recommendations.

The system does not place trades.

It analyzes historical performance and helps identify the strongest strategies for paper and future live testing.

## Current Features

- Multi-bot performance analysis
- Strategy ranking
- RUN / WATCH / PAUSE classification
- Historical review reports
- Capital allocation research
- AI Brain Strategy Advisor v1.4

## Active Strategies

- Simple Deep Buy
- Grid Trading
- Mean Reversion
- Binance Bot
- AI Signal Scanner
